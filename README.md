# Royal Hotel — Site Vitrine

Site web du **Royal Hotel**, Grand-Bassam, Côte d'Ivoire.

## Lancer le site (front-end)

**Prérequis :** Node.js 18+ installé

```bash
cd front/royal-hotel
npm install
npm run dev
```

Ouvre ensuite **http://localhost:5173** dans ton navigateur.

## Build de production

```bash
cd front/royal-hotel
npm run build
```

Les fichiers compilés se trouvent dans `front/royal-hotel/dist/`.
